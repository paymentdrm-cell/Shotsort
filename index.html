AI-Powered Photo Cataloging Tool

A personal project born from one simple frustration — thousands of disorganized photos with no way to find anything. Built end-to-end in Python using computer vision, machine learning, and SQL to automatically tag, catalog, and retrieve any photo in seconds.

01 — The Problem

Photography creates
a cataloging nightmare.

As an avid photographer, I accumulated thousands of images across multiple shoots, devices, and folders. Finding a specific photo meant scrolling endlessly through unnamed files like IMG_4872.jpg with zero context.

Existing tools like Lightroom require manual tagging — a tedious process most photographers skip entirely. Cloud solutions offered limited search, no color analysis, and no local control over your data.

I wanted to build something that could think like a photographer — understanding composition, lighting, subject, and palette — and organize an entire library automatically.

02 — The Approach

Three pillars of
intelligent organization.

Rather than a simple file renamer, the goal was a system that understood images the way a human would — by subject, mood, light, and place.

🧠

Computer Vision & AI Tagging
Used OpenCV and a pre-trained ResNet model via Scikit-learn to classify subjects — people, landscapes, architecture, food, events — with a confidence score, automatically, without any manual input.
🎨
02
Color Palette Extraction
Built a K-Means clustering algorithm in Python to extract the top 5 dominant colors from every image and store them as hex values, enabling search by mood, tone, and palette.
🗃️
03
SQL-Backed Metadata Engine

04 — Build Pipeline

From raw files to
intelligent library.

A six-stage automated pipeline processes every image from disk scan to fully cataloged, searchable record — zero human input required.

01
Photo Ingestion & EXIF Extraction
A Python script recursively scans a local directory, reads all image files (JPG, PNG, RAW), and extracts embedded EXIF metadata — camera model, aperture, shutter speed, ISO, GPS coordinates, and timestamp. Robust error handling catches missing or corrupted fields.
exif_data = piexif.load(img_path) → gps, timestamp, camera_model, iso
02
Subject Classification with ResNet
Each image is passed through a fine-tuned ResNet-based classifier via Scikit-learn's pipeline wrapper. The model outputs a primary subject tag — portrait, landscape, architecture, food, event, macro — with a confidence score stored per record.
classifier.predict(feature_vector) → ["landscape", 0.91]
03
Lighting Condition Analysis
OpenCV converts images to HSV color space and computes brightness and contrast histograms. Photos are labeled as golden hour, overcast, harsh midday, backlit, or low light — enabling photographers to filter by exact shooting conditions.
cv2.cvtColor(img, cv2.COLOR_BGR2HSV) → brightness_score, lighting_label
04
Dominant Color Palette Extraction
K-Means clustering (k=5) runs on pixel RGB values to extract the five dominant colors from each image. Stored as hex codes, enabling palette-based search — e.g., find all warm-toned golden hour photos in a single query.
KMeans(n_clusters=5).fit(pixels) → ["#D4842A", "#F2C46D", "#1A3A5C", ...]
05
SQL Database Population & Indexing
A normalized SQLite schema with tables for photos, tags, colors, and locations stores all pipeline outputs. Proper indexing on tag, date, and GPS fields ensures fast query performance across the full 4,200-photo library.
INSERT INTO photos (path, subject, lighting, palette_hex, lat, lng, shot_at)
06
Tableau Dashboard & Excel Export
Tableau connects live to the SQLite database to visualize shoot frequency by month, top subjects, color palette trends, and location heat maps. Pandas generates a clean 18-column Excel catalog for sharing with clients or collaborators.
pandas.to_excel(df, "catalog_export.xlsx") → 4,200 rows, 18 columns
05 — Key Features

What ShotSort can do.

🔍
Natural Language Search
Search your entire library in plain text — "golden hour portraits from last summer" — and get instant ranked results from the SQL index.
🗺️
Location-Based Grouping
GPS coordinates from EXIF data are parsed and grouped geographically, auto-creating albums by city, country, or custom radius.
🎨
Color Mood Filter
Filter photos by palette — warm, cool, monochrome, vibrant — using K-Means extracted dominant colors stored as searchable hex values.
☀️
Lighting Condition Labels
Every photo is scored and labeled by lighting type — golden hour, overcast, harsh sun, backlit, low light — helping recreate any look.
📊
Tableau Analytics Dashboard
A live dashboard shows shooting trends — most frequent subjects, monthly activity, top locations, and color palette evolution over time.
📁
Auto-Folder Organization
After classification, ShotSort optionally reorganizes your file system into a clean hierarchy by year → subject → lighting condition.
